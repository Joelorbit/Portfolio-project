# Eyuel Portfolio

A simple and clean personal portfolio website built with HTML, CSS, and JavaScript.

## Screenshot

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html` - Main page
- `eyu.css` - Styles
- `script.js` - Small interactions (menu, form message, B/W mode)
- `public/` - Images

## How To Run

1. Download or clone this project.
2. Open `index.html` in your browser.

## Main Features

- Responsive layout for desktop and mobile
- Hero section with profile image
- GitHub and Telegram icons in hero
- Contact form UI
- Black and white mode toggle

## Customize

- Update text in `index.html`
- Update colors in `eyu.css`
- Replace image in `public/`
