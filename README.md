Welcome to my **Portfolio Project!**

## 📂 Features

- **Responsive Design** – Looks stunning on all devices
- **Showcases Projects** – Interactive and visually appealing project gallery
- **Smooth Animations** – Engaging transitions and effects
- **Theme Customization** – Light/dark mode ready
- **Contact Form** – Easy way to reach out

## 📦 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Joelorbit/Portfolio-project.git
   ```
2. **Open `index.html` in your browser** (or use a local web server)

## 💡 Customization

- Update content in **index.html**
- Swap images in the **assets/images/** folder
- Tweak styles in **styles/main.css**
- Enhance interactivity in **scripts/main.js**
